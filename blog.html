Question-2: What is the difference between innerHTML, innerText, and textContent ?

Here's the difference between innerHTML, innerText, and textContent:
innerHTML: Gets or Sets the HTML markup inside an element.
innerText: Gets or Sets only visible text content of an element.
innerContent: Gets or Sets all text content within an element, including hidden text and script/style content.

Question-3: What is event delegation in the DOM?

Event delegation in the DOM is a technique in JavaScript where you attach a single event listener to a parent element to handle events triggered by its child elements, instead of attaching individual event listeners to each child.

Question-5: How do you create, add, and remove elements using JavaScript?

In JavaScript, you can create, add, and remove elements in the DOM dynamically using a variety of methods. Here's some example:
1.Creating Elements:
We can create new DOM elements using document.createElement().
2.Adding Elements:
